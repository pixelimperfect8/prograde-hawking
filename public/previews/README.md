# Gradient Preview Videos

This directory contains short video previews of each gradient mode for the landing page bento grid.

## Required Videos

Each mode needs a video file in both MP4 and WebM formats:

- `gradient.mp4` / `gradient.webm`
- `lava-lamp.mp4` / `lava-lamp.webm`
- `blob-stack.mp4` / `blob-stack.webm`
- `orbs.mp4` / `orbs.webm`
- `acid-trip.mp4` / `acid-trip.webm`
- `ripples.mp4` / `ripples.webm`
- `liquid-metal.mp4` / `liquid-metal.webm`
- `linear-gradient.mp4` / `linear-gradient.webm`
- `flow-gradient.mp4` / `flow-gradient.webm`
- `intelligence-glow.mp4` / `intelligence-glow.webm`

## How to Record

1. Open the app and select each gradient mode
2. Use screen recording software (OBS, QuickTime, etc.) to record 5-10 seconds
3. Crop the video to just the gradient area
4. Convert to MP4 (H.264) and WebM formats
5. Optimize for web (reduce file size, ~500KB per video ideal)
6. Name according to the mode ID (lowercase, hyphens instead of spaces)

## Specifications

- **Duration**: 5-10 seconds (looping)
- **Resolution**: 800x600 or similar (will be scaled to fit)
- **Format**: MP4 (H.264) and WebM (VP9)
- **File Size**: ~500KB or less per video
- **Framerate**: 30fps

## Fallback

If a video doesn't exist or fails to load, the CSS gradient animation will show instead.
